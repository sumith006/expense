Place your app icon PNG here at `assets/images/app_icon.png`.

Guidelines:
- Use a square PNG (recommended 1024x1024) with a transparent background.
- The launcher generator will create all required sizes for Android and iOS.

After you add your icon, run:

```bash
flutter pub get
flutter pub run flutter_launcher_icons:main
```
